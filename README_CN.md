# Remote Jobs - 微信小程序
连接远程机会，AI 助力简历精准匹配。
[English Version](./README.md)

## 核心亮点：AI 简历自动裁剪

本项目最大的特色是集成了强大的 **AI 自动裁剪** 技术。不再是千篇一律的投递，而是根据您看中的每一个岗位，由 AI 自动调整简历侧重点，确保您的经历与职位描述（JD）完美契合。

## 工业级工程化亮点

本项目遵循严苛的工业级前端开发标准，确保代码的可维护性、可扩展性及极致的用户体验。

- 💎 **标准化 UI 系统**: 引入完整的 **Design Token** 体系，通过 `app.wxss` 统一管理色阶、圆角与阴影，实现全应用视觉的高度一致性。
- 🛡️ **强制色彩规范**: 采用 **Stylelint** 自动化审核机制，严禁在 WXSS 中使用硬编码（Hardcoded）颜色代码，从架构层面杜绝 UI 碎片化。
- 🛠️ **全链路类型安全**: 基于 **TypeScript** 开发，对 API 响应、组件 Props 及全局状态进行严格类型定义。
- 🎭 **高阶状态机动效引擎**: 构建了深度解耦的登录墙动画系统，通过 CSS 变量插值与物理坐标投影技术，实现了跨层级组件在复杂鉴权生命周期下的毫秒级同步联动。
- 🖼️ **多模态 OCR 流程**: 深度集成 **Gemini Vision (2.5/3.0 Flash/Pro)**，支持拍照/截屏自动识别职位详情（JD），开启“截图即生成”的极致体验。
- 🌍 **深度国际化支持**: 内置成熟的 **i18n** 架构，支持中英文动态切换，覆盖日期、术语及 UI 语义。

## 功能特性

- ✂️ **一键 AI 裁剪**: 智能分析职位需求，自动改写简历内容，提升面试邀约率。
- 📸 **截图智能识别 (OCR)**: 基于 Gemini 多模态能力，瞬间将职位图片转化为结构化信息。
- 🔍 **精准职位发现**: 汇聚全球优质远程开发、设计、运营等职位。
- 📄 **多语言简历画像**: 构建专业的中英文简历，支持实时预览与编辑。
- 🖨️ **专业 PDF 导出**: 经过 AI 优化后的简历可直接导出为排版精美的 PDF。
- 📱 **原生流畅体验**: 采用微信小程序原生框架开发，响应速度快，交互体验佳。

## 技术栈

- **核心**: 微信小程序原生框架 + TypeScript
- **AI 能力**: Google Gemini 2.5/3.0 (Text & Vision)
- **工程化**: Stylelint (色彩规范校验), i18n Core, 自定义组件库
