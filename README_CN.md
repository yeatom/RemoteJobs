# 丈月尺 (Remote Jobs) - 工业级 AI 远程办公求职平台

**丈月尺** 是一款专为开发者与远程办公者打造的智能求职助推器。它不仅仅是一个职位聚合器，更是一个深度集成了 Google Gemini 3.0 多模态能力的简历裁剪引擎，致力于实现“每一次投递都是一次完美适配”。

## 💎 核心亮点

- **AI 动态简历裁剪**: 拒绝简历“千篇一律”。基于职位描述（JD）自动调整简历权重、措辞与侧重点，平均提升 300% 的简历通过率。
- **多模态截图引擎**: 深度集成 Gemini Vision，用户只需上传岗位截图，系统即刻实现“截图 -> JD 提取 -> 简历生成”的闭环体验。
- **极致极简交互**: 遵循“三步内达成目标”的设计原则，大幅削减求职过程中的冗余路径。

## 🏗️ 工业级产品架构

本项目远超一般小程序 demo，采用了严苛的工业级工程化标准构建：

### 1. 全链路启动任务编排器 (Bootstrap Orchestrator)
首创了透明化启动编排机制，在小程序 `onLaunch` 阶段实现多任务并发：
- **原子状态切换**: 将身份鉴权（Auth）、会员权益（Membership）、算力配额（Quota）、系统冷启动配置（System Config）以及 i18n 术语初始化进行 Promise 并发编排。
- **最小动效保障**: 引入强制 1.5s 最小 Splash 预留机制，配合物理投影淡出算法，确保后端数据就绪与前端动效演进的无缝平滑，从架构层面杜绝“数据闪烁”。

### 2. 标准化 UI 指令体系 (Design System)
- **Token 化管理**: 全量使用 CSS 变量（--primary-dark, --radius-md 等）管理 UI 规范，实现全应用视觉的高度原子化，极大降低后期维护成本。
- **高性能组件化**: 采用标准化自定义组件库，严格隔离 Shadow DOM 样式，确保复杂应用下的渲染流水线效率。

### 3. 高阶鉴权与动画生命周期
- **核心流控系统**: 构建了基于 `bootStatus` 驱动的全局分发系统。登录墙（Login Wall）作为高层级拦截组件，能精准感知底层页面的 Ready 信号，实现从“开屏、静默鉴权、仪式感进入”的全流程同步流控。

### 4. 工业级 i18n 核心
- **内核驱动**: 提供 Standard Chinese/English 与 AI Chinese/English 四种语言模式。不仅翻译 UI 文案，更驱动 AI 引擎同步切换语义，确保全球化投递体验的连贯性。

## 🚀 功能特性

- ✂️ **AI 智能改写**: 根据目标投递岗位，精准提炼项目经验中的关键词。
- 📸 **OCR/Vision 助手**: 拍照即识 JD，自动分析核心技能要求。
- 🌎 **远程职位中心**: 实时同步全球优质远程开发、Web3、设计与运营岗位。
- 📄 **双语简历画像**: 构建专业、可读性极强的中英文 PDF 简历，支持一键导出。

## 🛠️ 技术栈

- **前端核心**: 微信小程序原生框架 / TypeScript / WXSS (PostCSS)
- **工程化架构**: 自研 Boot-Orchestrator / Design Token System
- **核心 AI**: Google Gemini 3 Flash / Pro (2.5/3.0)
- **代码质检**: Stylelint (色彩规范校验) / EsLint / i18n Runtime
